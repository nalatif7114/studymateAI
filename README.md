# StudyMate AI

StudyMate AI adalah chatbot edukasi berbasis Artificial Intelligence (AI) yang menggunakan Google Gemini API untuk membantu pengguna mendapatkan informasi dan memahami materi pembelajaran secara interaktif.

## Fitur
- Tanya jawab berbasis AI
- Pemrosesan bahasa alami (NLP)
- Riwayat percakapan (Chat Memory)
- Antarmuka sederhana dan responsif
- Integrasi Google Gemini API

## Teknologi
- React.js
- Vite
- Express.js
- Google Gemini API
- JavaScript

### Backend
